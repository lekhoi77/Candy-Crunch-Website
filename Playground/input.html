Input Components

Design System with Data Attributes

Text Input

Basic Text Input (Medium Size)

HTML Code (Copy & Paste) 
<div class="input" data-type="text" data-state="default" data-size="medium">
  <label class="input-label">Label</label>
  <div class="input-field">
    <input type="text" placeholder="e.g. placeholder">
  </div>
</div>

With Optional Label

HTML Code (Copy & Paste) 
<div class="input" data-optional="true">
  <label class="input-label">Label</label>
  <div class="input-field">
    <input type="text" placeholder="e.g. placeholder">
  </div>
</div>

Dropdown / Select

Basic Dropdown

Select option

<div class="input" data-type="dropdown" data-size="medium">
  <label class="input-label">Choose an option</label>
  <div class="input-field">
    <div class="dropdown-trigger">
      <span class="dropdown-text">Select option</span>
      <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" class="dropdown-arrow">
        <path d="M18 9L12 15L6 9" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"/>
      </svg>
    </div>
    <div class="dropdown-menu">
      <button class="dropdown-option" data-value="1">Option 1</button>
      <button class="dropdown-option" data-value="2">Option 2</button>
      <button class="dropdown-option" data-value="3">Option 3</button>
    </div>
  </div>
</div>

Quick Reference

Data Attributes

  • data-type: "text" | "password" | "dropdown"
  • data-state: "default" | "focus" | "typed" | "error" | "disabled" | "open" | "picked"
  • data-size: "large" | "medium" | "small"
  • data-show-label: "true" | "false"
  • data-optional: "true" | "false"
  • data-checked: "true" | "false" (radio only)
